Fribbels HSR Scanner v1.0.3

  1. Open HSR and set language to English and screen resolution to 1920x1080
  2. Choose the non 'Full Screen' option. If it's the only option, hit Alt+Enter to exit full screen
  3. Open the 'Characters' menu
  4. Select a character that has all +15 relics
  5. Select the minimum enhance option below
  6. Click 'Start' and wait - do not move the mouse until done
  7. Press any key to cancel an in-progress scan
  8. Once finished, save your data to a file
